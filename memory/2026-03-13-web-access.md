1. **FDA搜索问题**:

   - 现状: 可以获取Guidance PDF，但搜索过滤不完全精确

2. **CDE网站问题**:
   - 原因: 网站使用复杂JavaScript混淆(类似反爬保护)
   - 现象: curl返回混淆的HTML，Playwright无法正常加载内容
   - 状态: 需要更高级的反检测措施或专门方案

3. **NMPA网站问题**:
