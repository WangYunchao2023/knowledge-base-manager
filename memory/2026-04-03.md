## guidance-web-access v3.9.24 重大修复（2026-04-03）

### FDA DataTables 专项修复
- **DataTables 检测**：`find_content_blocks` 中新增 `datatable_` case，自动识别 `#DataTables_Table_N` 表格
- **每页 100 条**：自动将 DataTables 分页调整为 100 条/页（从默认 10 条），页数从 279→28 页
- **PDF 精准过滤**：DataTables 链接中 `/media/.../download` 模式为 PDF 下载链接，每页 294 条 → 99 条
- **JS evaluate 翻页**：DataTables 分页按钮用 `page.evaluate('btn.click()')` 直接触发，避免 `element_handle.click` 超时
- **URL 参数降级**：当 `smart_interact_noexp` 检测到异常页面（apology/abuse）时，自动降级为 `?search=xxx` URL 参数搜索

### `cortana_auto_flow` 快速下载路径
- **final_download**：对于已经是完整 PDF 链接的结果（如 `fda.gov/media/.../download`），直接用 `page.goto + expect_download` 下载，跳过访问详情页
- **结果**：1991 个 FDA 指导原则 PDF，404 MB，~15 分钟完成

### 其他修复
- **页数限制**：DataTables 最多 10 页（1000 条），其他块最多 50 页（防止无限翻页）
- **修复 `or True` bug**：`safe_click_with_scroll_noexp` 中的 `page.evaluate(...) or True` 导致策略失败时不回退
- **HEADLESS 环境变量**：`HEADLESS=true python3 ...` 支持无头模式
- **`r_resp.body()`**：修复 Playwright `APIResponse.read()` → `.body()`
- **`getElementById` 兜底**：块选择器以数字开头时用 `getElementById` 代替 `querySelector`

### 经验分支 vs 无经验分支（重要）
- **经验分支**（`cortana_execute_flow`）：完全未动，保持原样
- **无经验分支**（`cortana_auto_flow`）：本次所有修复集中在这些函数：
  - `smart_interact_noexp`、`safe_click_with_scroll_noexp`
  - `find_content_blocks`、`find_next_button_for_block`
  - `get_links_by_text_content_v2`、`get_links_noexp`
  - `explore_with_pagination_noexp`、`execute_single_step`
  - `execute_strategies_batch`、`cortana_auto_flow`

### 版本号同步（必须同时更新三处）
1. `SKILL.md` frontmatter `version:`
2. `scripts/web_access.py` 顶部 `VERSION = "x.x.x"`
3. Git: `git add → git commit → git tag v{x.x.x}`
