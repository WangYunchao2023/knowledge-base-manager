
## CDE网站访问 - 经验总结

### 根本原因
1. **DISPLAY/XAUTHORITY丢失**：OpenClaw后台脚本启动浏览器时，必须带上：
   - `DISPLAY=:0`
   - `XAUTHORITY=/run/user/1000/gdm/Xauthority` (wangyc用户的认证文件)
   不然会报 "Missing X server or $DISPLAY"

2. **CDE搜索词策略**：搜索"沟通"比"沟通交流"更有效（前者17条，后者0条）

### 解决方案
```bash
DISPLAY=:0 XAUTHORITY=/run/user/1000/gdm/Xauthority python3 web_access.py ...
```

### 重要教训
- 遇到"Missing X server"不要尝试安装xvfb
- 先查memory/历史记录，比盲目试错快得多
- OpenClaw运行在有显示器的机器上时，后台脚本需要显式注入DISPLAY和XAUTHORITY
