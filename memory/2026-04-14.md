- **截图**：PlantUML/HTML 渲染 PNG（不支持 Mermaid 的图）
- **Mermaid**：简单流程图/时序图转 Mermaid（可直接插入飞书文本绘图）

### 脚本工具（v3.0.0 起）
- `scripts/render_to_png.py` — PlantUML/HTML 渲染 PNG（v3.0.0）
- `scripts/md_to_html.py` — .md 转 .html（v3.0.0，v3.0.1 修复离线渲染）
  - SVG 内嵌：PlantUML 用 `plantuml -TSvg` 本地渲染，SVG 内嵌 HTML，不需要网络
  - 支持类型：PlantUML / Mermaid / HTML / Vega / Infographic / Canvas
