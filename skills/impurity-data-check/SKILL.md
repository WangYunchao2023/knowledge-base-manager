# impurity-data-check

HPLC有关物质检测数据表明显错误检查工具。

## 检查项目

### 1. 批号格式一致性检查

同一文件内批号格式应统一，例如：
- `LMS002-26040105` vs `LMS002-260401-06` → 格式不统一
- 应统一为 `LMS002-260401-05` 或 `LMS002-26040105`

识别规则：
- 自动识别批号列（根据表头关键词）
- 检测连字符位置、数量不一致
- 8位日期码 vs 6位日期码+序号混用

### 2. 忽略不计判断合理性检查

判断逻辑：
- 标记"忽略不计"的含量值必须小于灵敏度溶液主成分峰面积的相当量
- 通常阈值：0.05%（或0.10%，根据各企业标准）
- 含量≥阈值却标"忽略不计" → 错误

## 使用方法

```bash
python3 ~/.openclaw/workspace/skills/impurity-data-check/scripts/checker.py <文件路径>
```

## 阈值配置

在 `checker.py` 中修改 `IgnoreTermChecker` 的 `threshold` 参数：
```python
ignore_checker = IgnoreTermChecker(threshold=0.10)  # 0.10% 阈值
```
