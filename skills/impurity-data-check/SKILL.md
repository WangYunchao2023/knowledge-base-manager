# impurity-data-check

HPLC有关物质检测数据表明显错误检查工具。

## 检查项目

### 1. 批号格式检查

常见问题：
- 批号缺少连字符（如 `LMS00226040105` 应为 `LMS002-260401-05`）
- 批号格式不统一（同一文件中混用多种格式）

识别规则：
- 自动识别批号列（根据表头关键词）
- 检测连字符缺失、位置错误等
- 同一文件内格式一致性比对

### 2. 忽略不计判断检查

判断逻辑：
- 标记"忽略不计"的含量值必须小于灵敏度溶液主成分峰面积的相当量
- 通常阈值：0.05%（或0.10%，根据各企业标准）
- 含量≥阈值却标"忽略不计" → 错误

## 使用方法

```bash
python3 ~/.openclaw/workspace/skills/impurity-data-check/scripts/checker.py <文件路径>
```
