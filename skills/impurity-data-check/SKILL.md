# impurity-data-check

HPLC有关物质检测数据明显错误检查工具。

## 架构

工具分为三个阶段：

```
1. 提取阶段：从 Excel 中提取项目关键信息（项目编号、药品名、产品名等）
2. 判断阶段：Agent 根据提取的信息 + 配置文件，综合判断使用哪个项目配置
3. 检查阶段：基于判断结果执行批号格式 + 忽略不计检查
```

**阈值优先级（从高到低）：**

| 优先级 | 来源 | 说明 |
|--------|------|------|
| 1 | `--threshold` 命令行参数 | 全局单一阈值 |
| 2 | `--agent-decision` Agent 决策 | Agent 直接指定配置 |
| 3 | `project_configs` 项目级配置 | 按项目编号/文件名/内容关键词匹配 |
| 4 | `default_threshold` / `impurity_thresholds` | 全局默认 |
| 5 | 内置默认值 | 0.05% |

## 两种工作模式

### 模式一：自动匹配（默认）

脚本根据项目编号（如 `LMS002`）自动匹配配置：

```bash
python3 scripts/checker.py <文件路径>
```

匹配顺序：项目编号（LMS002）→ 文件名关键词 → 内容关键词

### 模式二：Agent 协作（推荐）

```
Agent 检查流程：
  1. 使用 --extract-only 提取项目信息
  2. 查看可用配置列表
  3. 根据项目信息（项目编号、药品名等）判断使用哪个配置
  4. 使用 --agent-decision 指定决策结果
```

```bash
# Step 1: 提取信息（仅提取，不检查）
python3 scripts/checker.py <文件路径> --extract-only

# Step 2: Agent 根据信息判断，使用 --agent-decision 指定配置
python3 scripts/checker.py <文件路径> \
  --agent-decision '{"project": "黄体酮预灌封注射液", "threshold": 0.05, "impurity_thresholds": {"杂质M": 0.03, "其他单个杂质": 0.05}}'
```

## 配置文件

放置于 **skill 目录**：`~/.openclaw/workspace/skills/impurity-data-check/impurity-checker.yaml`

```yaml
# ── 项目级配置（支持项目编号和文字名称作为 key） ──
project_configs:

  # 黄体酮预灌封有关物质（项目编号：LMS002）
  "LMS002":
    default_threshold: 0.05
    impurity_thresholds:
      "杂质M": 0.03
      "其他单个杂质": 0.05

  "黄体酮预灌封注射液":
    default_threshold: 0.05
    impurity_thresholds:
      "杂质M": 0.03
      "其他单个杂质": 0.05

  # 可添加更多项目
  # "LMS001":
  #   default_threshold: 0.10
  #   impurity_thresholds:
  #     "杂质A": 0.10

# ── 全局默认配置（无项目匹配时使用） ──
default_threshold: 0.05
impurity_thresholds:
  "杂质M": 0.05
```

**匹配规则：**
- 项目编号（精确匹配，如 `LMS002`）→ 最优先
- 产品名/药品名（包含匹配）
- 文件名/路径关键词（包含匹配）

## 批号格式规范

标准格式：**项目编号-两位年两位月两位日-两位序号**

```
LMS002-260401-05
```

示例：
- `LMS002-260401-05` ✅ 合规
- `LMS002-26040105` ❌ 缺少连字符

## 检查项目

### 1. 批号格式一致性检查

- 检测同一文件内批号格式是否统一
- 识别缺少连字符的批号

### 2. 忽略不计判断合理性检查

- 标记"忽略不计"的含量值必须小于该杂质的报告限
- 含量 ≥ 该杂质的报告限却标"忽略不计" → 报错

## 使用方法

```bash
# 普通检查（自动匹配项目）
python3 scripts/checker.py <文件路径>

# 仅提取项目信息（不检查）
python3 scripts/checker.py <文件路径> --extract-only

# 指定 Agent 决策结果
python3 scripts/checker.py <文件路径> --agent-decision '{"project": "LMS002", "threshold": 0.05}'

# 指定全局阈值
python3 scripts/checker.py <文件路径> --threshold 0.06
```
