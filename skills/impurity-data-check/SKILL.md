# impurity-data-check

HPLC有关物质检测数据表明显错误检查工具。

## 批号格式规范

标准格式：**项目编号-两位年两位月两位日-两位序号**

```
LMS002-260401-05
 ↑        ↑↑↑↑↑↑  ↑
 项目编号  年月日    序号
```

示例：
- `LMS002-260401-05` ✅ 合规
- `LMS002-26040105` ❌ 缺少连字符
- `LMS002-260401-5` ❌ 序号位数不足

如需修改标准格式，编辑 `scripts/checker.py` 中的 `BatchFormatValidator` 类。

## 检查项目

### 1. 批号格式一致性检查

- 检测同一文件内批号格式是否统一
- 识别缺少连字符的批号（如 `LMS002-26040105` → 建议改为 `LMS002-260401-05`）
- 报错精准定位到单元格地址

### 2. 忽略不计判断合理性检查

- 标记"忽略不计"的含量值必须小于阈值
- 默认阈值：**0.05%**（可在 `scripts/checker.py` 中修改 `IgnoreTermChecker` 的 `threshold` 参数）
- 含量 ≥ 阈值却标"忽略不计" → 报错

## 配置

### 批号格式

在 `scripts/checker.py` 中修改正则表达式：

```python
# 默认格式：项目编号-年月日-序号
batch_format = BatchFormatValidator(
    pattern=r'^(.+)-(\d{2})(\d{2})(\d{2})-(\d{2})$',
    pattern_desc='项目编号-年月日-序号（例：LMS002-260401-05）'
)
```

### 忽略不计阈值

```python
# 默认阈值：0.05%
errors = check_file(file_path, ignore_threshold=0.05)
```
