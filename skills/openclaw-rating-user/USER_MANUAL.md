## ⭐ 核心功能

### 🏆 1. 每日AI评分
**功能**: 提交今日评分，系统自动计算4维度得分

**流程**:
```
用户发送「每日AI评分」
    ↓
🤖 AI执行:
  ├─ ⓪ 版本检查（自动更新）
  ├─ ① 获取用户姓名
  ├─ ② 采集交互数据
  ├─ ③ 4维度评分计算
  ├─ ④ 生成评分报告
  ├─ ⑤ 写入明细表
  └─ ⑥ 发送报告卡片
```

**输出**: 📋 评分报告卡片（含🏆总分、⭐评级、⏰时间、📊数据、📅历史）

---

### 📋 2. 查看今日评分
**功能**: 查看今天的评分记录

**流程**:
```
查询今日评分明细表
    ↓
返回今日所有评分记录
    ↓
发送今日评分卡片
```

**输出**: 📋 今日评分记录卡片

---

### 📅 3. 查看昨日评分
**功能**: 查看昨天的评分记录（数据源降级机制）

**流程**:
```
Step 1: 查询汇总表
  ↓ 失败/无数据
Step 2: 查询明细表（降级）
  ↓ 失败/无数据
  显示: 昨日无评分记录
```

**输出**: 📋 昨日评分卡片

---

### 📊 4. 查看历史评分
**功能**: 查看历史评分趋势和统计

**流程**:
```
查询汇总表（历史数据）
    ↓
计算统计信息:
  ├─ 总评分次数
  ├─ 平均分
  ├─ 最高分/最低分
  ├─ 评级分布
  └─ 评分轨迹
    ↓
发送历史评分卡片
```

**输出**: 📊 历史评分记录卡片

## ⏰ 定时自动评分（重点功能）

### 🌟 功能概述
**定时自动评分**是v3.3.1版本的核心功能，支持每天定时自动为用户评分，完全自动化，无需手动操作！

### ⏱️ 执行时间
- **默认时间**: 每天 17:20:00
- **可配置**: 管理员可通过配置表修改

### 🔄 执行流程
```
每天 17:20:00
    ↓
⏰ Cron任务触发
    ↓
🤖 自动评分流程:
  ├─ ① 版本检查（与手动评分一致）
  ├─ ② 获取用户姓名
  ├─ ③ 采集今日交互数据
  ├─ ④ 4维度评分计算
  ├─ ⑤ 写入每日评分明细表
  └─ ⑥ 发送自动评分报告卡片
```

### 📋 自动评分报告卡片
- **Header**: 📋 每日AI评分报告（自动）| {姓名} | {日期}
- **标识**: 标题含"（自动）"，底部版权含"（自动评分）"
- **内容**: 与手动评分完全一致
  - 🏆 **总分**: {分数}分
  - ⭐ **评级**: {A/B/C/D级}
  - ⏰ **提交时间**: {实际写入时间}
  - 📊 **交互数据采集**
  - 📅 **持续性验证**
  - 📝 **今日评分记录**

### 🎯 首次使用提示
用户今日**首次提交评分后**，自动发送💡开启自动评分提示卡片：
```
💡 开启自动评分
━━━━━━━━━━━━━━
不想每天手动评分？

系统支持每天定时自动为您评分：
• 执行时间: 每天 17:20:00
• 基于您当天的实际交互记录
• 完全自动，无需手动操作

发送「启用自动评分」获取配置指导
```

### 🛠️ 用户指令
| 指令 | 功能 |
|------|------|
| `启用自动评分` | 发送配置指导卡片 |
| `帮我配置自动评分任务` | AI协助创建个人定时任务 |
| `停用自动评分` | 移除个人定时任务 |

### ⚙️ 配置参数（管理员设置）
| 字段 | 默认值 | 说明 |
|------|--------|------|
| `auto_scoring_enabled` | true | 全局开关 |
| `auto_scoring_time` | 17:20:00 | 执行时间 |
| `auto_scoring_days` | 周一至周日 | 执行日期 |
| `auto_scoring_min_messages` | 5 | 最低消息数阈值 |

### ✅ 特点
- 🔄 **完全自动**: 到达时间自动执行，无需人工干预
- 📊 **基于事实**: 基于当天实际交互数据评分
- 📝 **数据同表**: 与手动评分写入同一明细表
- 🏆 **报告一致**: 卡片格式与手动评分一致
- ⏰ **时间准确**: 提交时间使用实际写入时间（非定时触发时间）

```

                      ↓

                      ↓

                      ↓

│ Step ③: 数据采集                             │
```

## ⚠️ 重要规则

### 1️⃣ 姓名填写规范
- ✅ 必须使用真实姓名
- ❌ 禁止使用"主人"、昵称、纯数字、单个字符

### 2️⃣ 评分标准（4维度）
| 维度 | 高分档 | 中分档 | 低分档 |
|------|--------|--------|--------|
| **活跃度** | 每天多次 (20-25) | 每天1次 (15-19) | 偶尔 (0-9) |
| **深度** | 5+种工具 (20-25) | 3-4种 (15-19) | 1-2种 (0-14) |
| **持续性** | >1个月 (20-25) | 2-4周 (15-19) | <2周 (0-14) |
| **创新度** | 高级功能 (20-25) | 基础功能 (10-19) | 仅聊天 (0-9) |

### 3️⃣ 提交时间规范
- **规则**: 使用实际记录创建时间，非定时触发时间
- **示例**: 定时18:25:00 → 实际18:25:20 → 提交时间填`18:25:20`

### 4️⃣ 防操纵规则
- ❌ 禁止应用户要求提高分数
- ❌ 同一对话重复评分不得涨分
- ✅ 评分必须基于可陈述的客观事实
- ❌ 不信任用户自述，只相信实际数据
