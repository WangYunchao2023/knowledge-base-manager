# check-inspection-data

> 跟检数据错误自动检查工具——检测批号格式不规范、日期填写错误等常见问题。

## 安装依赖

```bash
pip install openpyxl pyyaml
```

## 快速开始

### 方法1：命令行直接运行

```bash
python scripts/checker.py /path/to/your/file.xlsx
```

### 方法2：通过 skill 调用

```markdown
/check-inspection-data /path/to/your/file.xlsx
```

## 检查范围

| 类别 | 检查项 |
|------|--------|
| **批号格式** | 长度异常、非法字符、疑似写反 |
| **日期检查** | 无效日期（不存在）、格式不统一、超范围、逻辑矛盾 |
| **通用检查** | 必填字段为空、数值超范围 |
