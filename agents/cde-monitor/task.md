# CDE指导原则检查任务

## 任务目标
每天早晨检查CDE官网是否有新发布的指导原则和征求意见稿。

## 任务步骤

### 1. 分层下载策略（推荐）

采用分层降级策略，确保任务稳定完成：

#### 第1层：优先使用 cde-download 命令
```bash
python3 ~/.openclaw/workspace-cortana/skills/web-access/scripts/web_access.py cde-download "https://www.cde.org.cn/main/news/viewInfoCommon/xxx"
```
- **目标**：发现需要下载的内容时，直接用 cde-download 尝试下载
- **下载失败标准**：命令返回错误、或下载的文件 `file` 命令验证不是有效PDF

#### 第2层：若第1层失败，继续使用 web-access skill 其他命令
| 命令 | 作用 |
|------|------|
| `pdf <url>` | 查找页面中所有PDF链接 |
| `links <url>` | 获取页面所有链接 |
| `download <url>` | 从已知链接下载文件 |

- **触发条件**：第1层下载失败后，自动继续尝试
- **无需人工判断**，直接按顺序尝试其他命令

#### 第3层：若 web-access skill 全部命令都不行，使用 browser 点击下载
```python
browser navigate to https://www.cde.org.cn/main/news/viewInfoCommon/xxx
# 找到下载链接并点击
```
- **最后备用**：确保任务一定能完成
- **适用**：web-access skill 所有方法都无法成功下载时

### 2. 验证发布日期
- **进入每个指导原则的详情页**
- **确认实际发布日期是否与列表显示一致**
- 若有偏差，以详情页显示的日期为准进行修正
- 注意：有时列表显示的日期可能不准确

### 3. 下载保存
- 保存目录: ~/Documents/工作/指导原则更新/
- 文件名格式: "发布日期 - 文件名.pdf"
- **如果下载的是Word文档，先保存，然后转换为PDF，最后删除Word原文件**
- **只保存PDF版本**
- **必须用`file`命令验证是真实PDF**（不是加密HTML）

### 4. 文档格式转换
- **如果下载的不是LLM支持的格式**（如ceb等）
- **自行转换为LLM支持的格式**：
  - PDF：直接支持
  - Word：直接支持
  - 其他格式：转换为PDF或Word
- 确保后续AI分析能正常读取

### 5. AI分析
使用PDF分析工具读取内容，提取：
- 新要求（以前没有的）
- 与以往要求不同的内容
- 与从业人员普遍认知不同的内容
- 涉及药品研发、生产、注册的重大变化

**重要信息标注要求**：
- **每个重要信息点必须标注页码**（如：P12 表示第12页）
- 页码信息格式：`(P页码)`
- 如果是多个页码，用逗号分隔（如：P3, P15-17）
- **在原文PDF/Word中对重要内容进行标黄标注**

### 6. 生成Word文档
- 模板位置：agents/cde-monitor/reference/摘录模板.docx
- **每次运行前检查模板是否有更新**，如有更新重新读取格式
- 每月一个文档
- 命名格式: "YYYY年MM月 - 指导原则摘录.docx"
- 追加格式（严格按模板格式）:
  * 日期行：Times New Roman, 18pt, 加粗, 黄色背景
  * 标题行：Times New Roman, 15pt, 加粗, 浅绿色背景
  * 内容摘录：加粗
  * 重要信息：红色，11pt，加粗，**必须标注页码如(P12)**

### 7. 飞书通知
- **若有新发布**：
  - 在飞书channel中发送摘录内容
  - 格式与Word文档一致，包含页码信息
- **若无新发布**：
  - 在飞书channel中汇报："本期无新发布"
- 飞书channel配置：可在cron任务中设置

## 输出要求
完成后报告：
- 是否发现新发布
- 下载了几个文件
- 文档格式转换情况（如有）
- Word文档更新情况
- 飞书通知发送情况

## 重要原则
- **优先**使用 web-access skill 的 cde-download 命令
- **降级**使用 web-access 其他命令（pdf/links/download）
- **最后**使用 browser 点击下载作为备用
- **不要**使用 curl/wget 直接下载（会得到加密HTML）
- **必须验证**发布日期的准确性
- **确保**文档格式可被LLM读取
- **必须**在飞书channel进行通知（若配置）
- **重要信息**不限字数，在简洁的前提下说明白
- **必须**标注页码，方便快速定位查阅
