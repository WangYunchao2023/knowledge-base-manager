# 指导原则摘录模板

## 文档格式

每月一个Word文档，命名格式：`YYYY年MM月 - 指导原则摘录.docx`

## 页面设置

- 纸张：A4
- 上/下边距：2.54cm
- 左/右边距：3.175cm

## 行距

- **1.5倍行距**（应用于整个文档）

---

## 单条格式

### 1. 日期行
```
2026-03-11
```
- 字体：Times New Roman
- 字号：**18pt**
- 加粗：是
- 颜色：默认（黑色）
- **背景色：黄色 (FFFF00)**

### 2. 标题行
```
1、流感病毒疫苗临床试验技术指导原则（试行）
```
- 字体：Times New Roman
- 字号：**15pt**
- 加粗：是
- 颜色：黑色 (000000)
- **背景色：浅绿色 (81D41A)**

### 3. 内容摘录标题
```
内容摘录：
```
- 字体：Times New Roman
- 字号：默认（11pt）
- 加粗：是
- 颜色：默认

### 4. 内容摘录正文
```
本指导原则旨在为流感病毒疫苗的临床试验提供技术指导，包括试验设计、主要终点、安全性评价等关键要求...
```
- 字体：Times New Roman
- 字号：默认（11pt）
- 加粗：否
- 颜色：默认

### 5. 重要信息标题
```
重要信息：
```
- 字体：Times New Roman
- 字号：**11pt**
- 加粗：是
- 颜色：**红色 (FF0000)**
- 段后：0.9cm（约127000 EMU）

### 6. 重要信息列表

- 字体：Times New Roman
- 字号：默认（11pt）
- 加粗：否
- 颜色：默认
- 段后：0.9cm
- **必须标注页码：格式为 (P页码)，如 P12 或 P3, P15-17**

## 示例

**2026-03-11** （黄色背景）

1、流感病毒疫苗临床试验技术指导原则（试行） （浅绿色背景）

内容摘录：
本指导原则旨在为流感病毒疫苗的临床试验提供技术指导，包括试验设计、主要终点、安全性评价等关键要求...

重要信息：
- 新增：首次明确流感疫苗与其他疫苗同时接种的免疫原性评价要求 (P12)
- 变化：调整了III期临床试验样本量计算方法 (P8, P15-17)
- 重要：要求在至少3个不同地理区域开展多中心试验 (P5)

2、罕见病酶替代疗法药物药学研究与评价技术指导原则（征求意见稿） （浅绿色背景）

内容摘录：
本指导原则针对罕见病酶替代疗法药物的药学研究与评价提出技术要求...

重要信息：
- 新领域：国内首个酶替代疗法专门指导原则 (P3)
- 关注点：产品稳定性要求、酶活性检测方法 (P7, P10-12)

## 重要信息标注规则

| 标注 | 含义 |
|------|------|
| 新增 | 以前没有的新要求 |
| 变化 | 与以往要求不同的内容 |
| 重要/关注点 | 特别值得关注的重大变化 |

### 页码标注

- **每个重要信息点必须标注页码**
- 格式：`(P页码)`
- 示例：`(P12)` 或 `(P3, P15-17)`
- 方便读者快速定位原文进行查阅
- **同时在原文PDF/Word中对重要内容进行标黄**

### 内容要求

- **重要信息不限字数**
- 根据重要性，在简洁的前提下说明白
- 确保科研人员能理解核心变化
